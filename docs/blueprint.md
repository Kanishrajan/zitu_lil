# **App Name**: ZITU

## Core Features:

- Themed UI: Implement a black and white themed user interface inspired by Uber's design principles, focusing on clean aesthetics and high readability.
- Content Feed: Implement an Instagram-style scrolling feed with posts containing images, text, and videos to facilitate content sharing and networking.
- Business Discovery: Enable users to search and filter businesses by category, region, and trending status to facilitate targeted business discovery.
- Marketplace: Allow businesses to list products/services with detailed information, enabling other users to place bids or negotiate offers directly within the app.
- Networking: Facilitate networking by allowing users to create profiles, follow other users, and engage in real-time chat.

## Style Guidelines:

- Primary background color: Black (#000000) to provide a sleek and modern base.
- Secondary background color: White (#FFFFFF) for content areas and clear text contrast.
- Gray accents: Dark Gray (#121212) and Light Gray (#E0E0E0) for subtle UI elements and borders.
- Highlight color: Electric Blue (#1E90FF) for interactive elements and calls to action, providing visual cues.
- Font: 'Inter' (sans-serif) for a minimal and highly readable text throughout the app.
- Utilize full-width buttons with rounded corners (10px) to ensure clear call-to-actions on black and white backgrounds.
- Incorporate smooth transitions and animations for a fluid, Uber-style user experience within the black-and-white theme.
